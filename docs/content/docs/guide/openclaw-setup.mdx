本指南帮助 OpenClaw（龙虾）用户将 Agent 接入 Nebula API 中转站。

## 前提条件

- 已注册 Nebula API 账号
- 已获取 API Key（参考 <a href="/docs/guide/quickstart/">快速开始</a>）

## 接入步骤

### 1. 编辑配置文件

打开 OpenClaw 配置文件：

```bash
~/.openclaw/openclaw.json
```

### 2. 添加自定义 Provider

在配置文件中添加以下内容：

```json
"models": {
  "providers": {
    "nebula": {
      "baseUrl": "https://ai.nebulatrip.com",
      "apiKey": "你的API Key",
      "api": "anthropic-messages",
      "models": [
        {
          "id": "claude-opus-4-5",
          "name": "Claude Opus 4.5"
        },
        {
          "id": "claude-opus-4-6",
          "name": "Claude Opus 4.6"
        },
        {
          "id": "claude-sonnet-4-5",
          "name": "Claude Sonnet 4.5"
        },
        {
          "id": "claude-sonnet-4-6",
          "name": "Claude Sonnet 4.6"
        }
      ]
    }
  }
}
```

### 3. 设置 Agent 模型

将模型绑定到你的 Agent：

```bash
openclaw models set nebula/claude-opus-4-6 --agent 你的agentID
```

### 4. 重启 Gateway

```bash
openclaw gateway restart
```
