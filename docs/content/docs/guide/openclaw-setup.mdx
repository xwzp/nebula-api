本指南帮助 OpenClaw（龙虾）用户将 Agent 接入 Nebula API 中转站。

## 前提条件

- 已注册 Nebula API 账号
- 已获取 API Key（参考 [快速开始](/docs/guide/quickstart)）

## 接入步骤

### 1. 编辑配置文件

打开 OpenClaw 配置文件：

```bash
~/.openclaw/openclaw.json
```

### 2. 添加自定义 Provider

在配置文件中添加以下内容：

```json
"models": {
  "providers": {
    "nebula-anker": {
      "baseUrl": "https://ai.nebulatrip.com",
      "apiKey": "你的API Key",
      "api": "anthropic-messages",
      "models": [
        {
          "id": "claude-opus-4-5",
          "name": "Claude Opus 4.5"
        },
        {
          "id": "claude-opus-4-6",
          "name": "Claude Opus 4.6"
        },
        {
          "id": "claude-sonnet-4-5",
          "name": "Claude Sonnet 4.5"
        },
        {
          "id": "claude-sonnet-4-6",
          "name": "Claude Sonnet 4.6"
        }
      ]
    }
  }
}
```

### 3. 设置 Agent 模型

将模型绑定到你的 Agent：

```bash
openclaw models set nebula-anker/claude-opus-4-6 --agent 你的agentID
```

### 4. 重启 Gateway

```bash
openclaw gateway restart
```

## 常见问题

### API Key 在哪里获取？

登录 Nebula API 后台，进入 <a href="/console/token">控制台 → 令牌管理</a> 创建新令牌。

### 支持哪些模型？

目前支持 Claude Opus 4.5/4.6 和 Claude Sonnet 4.5/4.6。具体可用模型请登录控制台查看。

### 请求报错 401？

请检查 API Key 是否正确，以及令牌是否有足够的额度。
